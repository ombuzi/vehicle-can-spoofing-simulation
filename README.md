# Vehicle CAN Spoofing Simulation (Kali Linux)

This project simulates CAN bus message spoofing in a virtual environment using vcan0 on Kali Linux.

## Features Simulated
- 🚪 Door lock spoofing
- 🛑 Emergency braking spoof
- ⚠️ Throttle cutoff attack
- 🕵️ CAN bus sniffing

## Tools Used
- can-utils
- vcan0 (virtual CAN)
- bash scripting
